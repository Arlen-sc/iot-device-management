## 快速开始

### 1. 编译项目
```bash
mvn clean package -DskipTests
```

### 2. 运行应用
```bash
java -jar target/iot-device-management-1.0.0-SNAPSHOT.jar
```

### 3. Docker运行
```bash
docker-compose up -d
```

### 4. 访问系统
- 应用地址: http://localhost:18082
- 登录页面: http://localhost:18082/login.html
- Swagger文档: http://localhost:18082/swagger-ui.html
- 健康检查: http://localhost:18082/actuator/health

### 5. 默认账号
- 用户名: admin
- 密码: admin123
